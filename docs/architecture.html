3. AOP (Aspect-Oriented Programming)

في Laravel نطبّق AOP عبر:

  • ConcurrencyAspect — نقاط قطع (join points) حول العمليات الحرجة
  • ConcurrencyTracingMiddleware —切面 على كل طلب HTTP
  • تسجيل metrics: مدة، نجاح، فشل لكل نقطة تزامن
ConcurrencyAspect::around('inventory.pessimistic_lock', fn () => ...);
ConcurrencyAspect::around('payment.simulated_gateway', fn () => ...);

نقاط التزامن (Synchronization Points)

  • قفل صف المنتج — products pessimistic
  • قفل رصيد المستخدم — users pessimistic
  • عداد السعة — Cache atomic
  • Circuit breaker state — Cache
